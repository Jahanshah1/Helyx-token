# Helyx-token

currently deployed on ropsten testnet - 0xf90584d35d301339d817518c825c42514f31f489
- mintable = True
- burnable = True
- votes = True 
- Transfer of Ownership = True

## overview 
Helyx (HLX) is an `ERC-20` token deployed on Ropsten test-net with on-chain governance/voting capabilities. which can be used for micro transactions on blockchain and voting for any upcoming changes or ideas. This smart contract can be further enhanced with integrating an UI for making the voting system more viable for holders.
